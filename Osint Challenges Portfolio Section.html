Challenge #1 GEOINT Location Identification

Objective: Identify a location from visual and contextual clues.

Techniques: GEOINT, visual analysis, environmental clue extraction, location narrowing.

Sources & Tools: Reverse image search, Google Maps, Street View, AI-assisted location filtering.

Challenge #2 Email Attribution via Repository Analysis

Objective: Trace the origin of an OSINT tool creator’s public email footprint.

Techniques: GitHub repository review, commit history analysis, timestamp interpretation.

Sources & Tools: GitHub, public code repositories, commit metadata, timestamp conversion.

Challenge #3 Public Record & Identity Correlation

Objective: Link public identifiers to a target profile using open-source records.

Techniques: Name-based search, public-record review, cross-source validation.

Sources & Tools: Search engines, public records, social media review, breach-checking platforms.

Challenge #4 Company Records & Address Research

Objective: Identify historical address information from company-related open records.

Techniques: Corporate registry analysis, document review, timeline matching.

Sources & Tools: French company records, public filings, registry search platforms.

Challenge #6 Historical Event & Social Footprint Research
