OSINT Challenges & Investigation Methodology

This section summarizes OSINT challenges completed to strengthen practical investigation skills across GEOINT, cybercrime attribution, public-record research, archive analysis, breach-data correlation, and digital footprint tracing.

All work was conducted using passive, legal open-source intelligence methods. No private access, account interaction, harassment, or restricted-system access was used. Sensitive answers, personal identifiers, and screenshots are intentionally excluded.

Challenge #1

GEOINT Location Identification

GEOINT Street View Visual Analysis

Objective: Identify a real-world location from an image-based clue.

Tools Used: Google Maps, Google Street View, reverse image search, AI-assisted location narrowing.

How I approached it: I reviewed visible environmental clues such as road layout, building height, commercial signs, street structure, and nearby transport indicators. I then narrowed possible areas and validated the match using Street View comparison.

Challenge #2

Email Attribution via GitHub Repository Analysis

GitHub Commit History Metadata

Objective: Trace the public email footprint of an OSINT tool creator.

Tools Used: GitHub, repository history, commit records, public profile review, timestamp conversion.

How I approached it: I reviewed the creator’s GitHub repositories, checked older commits and documentation files, and compared email appearances across project history to identify the earliest public usage timeline.

Challenge #3

Public Record & Identity Correlation

Public Records Search Engines Correlation

Objective: Link open-source identity indicators to a target profile.

Challenge #12

Cybercriminal Forum Attribution

Wayback Machine Google Dorking Cross-Platform Pivoting

Objective: Attribute a historical forum post to an online identity using open-source evidence.

Tools Used: Wayback Machine, Google dorking, public breach datasets, Have I Been Pwned, YouTube, Twitter/X, archived forum pages.

How I approached it: I reconstructed the forum thread using archive data, identified username reuse, pivoted from username to email, then correlated public platform activity to assess common ownership and attribution confidence.
